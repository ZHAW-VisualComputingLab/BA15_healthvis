Einleitung Grippedaten

Folgend werden Daten zu der Infektionskrankheit Influenza visualisiert. Die Daten stammen aus den Datenquellen OpenData, Google Trends und dem BAG.

  • Daten aus OpenData stellen die Mortalität aufgrund dieser Infektion dar
  • Daten von Google Trends stellen die Suchbegriffe zum Thema Influenza dar
  • Daten von dem BAG stellen die gemeldeten Fälle von Influenza dar

Bedienhinweis: Mit einem Mausklick, auf die, mit einem schwarzen Pfeil gekennzeichneten Bedienelemente (links und rechts auf der Seite), kann eine Seite weiter- bzw. zurückgeschaltet werden.

Zusätzlich kann direkt auf eine spezielle Seite gewechselt werden. Dies ist möglich mit einem Mausklick auf die sich unten in der Mitte befindenden Kreise. Die aktuelle Seite hat einen weissen Kreis, die anderen einen Schwarzen.
